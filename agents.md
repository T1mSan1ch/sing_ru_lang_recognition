# Agent: nonverbal-sign-language-service

## 1. Описание проекта

Проект `nonverbal` — это backend-сервис на Python 3.12 + FastAPI для распознавания русского жестового языка (РЖЯ) из видео.

Архитектура решения:

1. CV-модель (из проекта Slovo) распознаёт отдельные жесты → возвращает последовательность слов.
2. LLM-модуль агрегирует последовательность слов → формирует осмысленные предложения.
3. FastAPI предоставляет REST API для клиентов.
4. Сервис будет развернут в Yandex Cloud с использованием GPU.

Источник CV-модели:
https://github.com/hukenovs/slovo

## 5. CV-модель (Slovo)

Модель должна:

- Загружаться один раз при старте приложения
- Работать на GPU (если доступен)
- Поддерживать инференс батчами
- Возвращать:
  - список слов
  - confidence score

Предобработка видео должна быть вынесена отдельно.

---

## 6. LLM-агрегатор

Назначение:
Преобразовать список слов:

["привет", "я", "идти", "магазин"]

в:

"Привет, я иду в магазин."

Требования:
- Асинхронный вызов
- Таймаут
- Fallback: join(words)
- Возможность заменить провайдера LLM
